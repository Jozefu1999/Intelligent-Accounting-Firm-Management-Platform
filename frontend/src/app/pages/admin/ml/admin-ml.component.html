Operational notes

  • Verify project and client data is up to date before retraining.
  • Run retraining during low-activity windows.
  • Set the DB_URL environment variable to train on real database data.
  • Monitor accuracy after each execution — a drop may indicate data drift.
  • Both models fall back to synthetic data when real data is insufficient.
